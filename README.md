# 💰 Financastz

## 📌 Descrição do Projeto

O **Financastz** é uma aplicação web voltada para gestão financeira pessoal, permitindo que usuários acompanhem suas receitas, despesas e tenham maior controle sobre sua vida financeira.

---

## 🌐 Acesse no navegador

[Finanças_Tz](https://financastz.base44.app/)

---

## 🚀 Como rodar o projeto no vscode

### 📦 1. Clonar o repositório

git clone https://github.com/seu-usuario/financastz.git  
cd financastz

---

### 📥 2. Instalar dependências

Se estiver usando Node.js:

npm install  

ou

yarn install  

---

### ▶️ 3. Rodar o projeto em ambiente de desenvolvimento

npm run dev  

---


## 🛠️ Tecnologias Utilizadas

Frontend:

- HTML5  
- CSS3  
- Vite  

Backend / Plataforma:
- Base44
